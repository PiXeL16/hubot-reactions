# hubot-reactions
![hubot-reactions](https://i.imgur.com/Ie17FZ6.gif)

More than 100 awesome reactions GIFs to compliment your teammates

## Instalation
```
$ npm install --save hubot-reactions
```
Add `["hubot-reactions"]` to your `external-scripts.json` file.
